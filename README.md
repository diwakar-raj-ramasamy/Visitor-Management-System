# Visitor Management System (VMS)

## Overview

The Visitor Management System (VMS) is a web-based application developed to simplify and automate visitor handling in an organization. It helps receptionists, employees, administrators, and visitors manage the entire visitor process digitally instead of using manual registers.

The system allows visitors to register their details, request meetings with employees, receive approval, and generate visitor passes. Receptionists can manage check-in and check-out activities, while administrators can monitor visitor records and generate reports.

## Problem Statement

Traditional visitor management relies on paper registers, which can be time-consuming, difficult to maintain, and prone to errors. Finding past visitor records and tracking visitor movement becomes challenging.

This project provides a digital solution that improves security, efficiency, and record management.

## Objectives

* Automate visitor registration.
* Reduce paperwork and manual effort.
* Improve visitor tracking and security.
* Generate visitor passes digitally.
* Provide role-based access for different users.
* Maintain visitor history and reports.

## System Workflow

1. Visitor registers in the system.
2. Visitor submits a pass request.
3. Employee receives the request.
4. Employee approves or rejects the request.
5. Receptionist verifies the approval.
6. Visitor checks in and receives a visitor pass.
7. Receptionist records visitor check-out.
8. Admin can view reports and visitor history.
